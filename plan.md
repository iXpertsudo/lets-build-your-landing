# Restaurant Website Plan

Build a complete, appetizing restaurant website with a warm color scheme and high-quality food photography.

## 1. Components & Architecture
- `Navbar`: Sticky, responsive navigation with logo and "Book Now" CTA.
- `Hero`: High-impact section with mouth-watering image, headline, and reservation shortcut.
- `SpecialOffers`: Framer motion carousel showcasing current deals.
- `Menu`: Interactive component with categories (Starters, Mains, Desserts, Drinks) and dietary filters (Vegan, Gluten-Free, Spicy).
- `ChefProfiles`: Section highlighting the culinary team.
- `ReservationSystem`: Form for booking tables (Date, Time, Guests, Contact).
- `Reviews`: Customer testimonial section.
- `EventBooking`: Information and form for private events/catering.
- `Location`: Contact info + embedded Google Map placeholder.
- `Footer`: Links, social media, and opening hours.

## 2. Technical Stack
- React 19, Vite, Tailwind CSS (using Amber/Warm palette).
- `lucide-react` for iconography.
- `framer-motion` for smooth transitions and carousel.
- `sonner` for form submission notifications.
- `shadcn/ui` for forms, buttons, and input components.

## 3. Visual Design (UI/UX Pro Max)
- **Palette**: Warm Amber & Cream (#FFFBEB background, #92400E primary, #451A03 text).
- **Typography**: Playfair Display (Headings) + Inter (Body).
- **Style**: Premium Editorial.
- **Imagery**: Professional food shots from Unsplash.

## 4. Implementation Steps
1. Generate high-quality food and restaurant images.
2. Setup Tailwind theme in `index.css`.
3. Create modular components.
4. Implement interactive menu filtering logic.
5. Build reservation and event forms with validation.
6. Assemble the main page in `App.tsx`.
7. Final polish and build validation.
